# human_list.py 機能
```
ファイルに記載された数名の特徴を指定し出力できるコード
python human_list.py -m (特徴)＊複数可の入力で
["BMI","身長","体重","足のサイズ","職業","職歴"]
これらの特徴の公開と設定しているデータを呼び出せる

追加機能として
python human_list.py -h で
"Hello" の出力が可能
```
## コマンドメモ
```
身長など特徴のリクエスト
<p>python human_list.py -m 身長　(例)</p>
あいさつ
<p>python human_list.py -h</p>
```
